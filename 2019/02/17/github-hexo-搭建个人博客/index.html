    github + hexo 搭建个人博客

    terryrao
    2019-02-17

    使用github pages 和 hexo 搭建一个个人博客网站。github pages 是 github 公司提供的免费的静态网站托管服务。从主页进去我们可以看到它主要有两个用途：作为个人或者组织的站点或者作为项目的站点，因此我们可以用这个服务来搭建自己的博客，使用 jekyll 或者 hexo 等工具可以创建自己风格的主题。

    1. github pages

    搭建过程在 github pages 上有详细的介绍，首先注册一个 github 账号，然后在自己的电脑里下载 git 。

    1. Windows

      Windows 系统下可以下载 github desktop 也可以下载其它的 git 工具，不过由于其它原因可能比较慢，最好是使用代理或者在国内的云盘如百度云里找个可用的安装包来安装。

    2. Mac

      Mac 用户可以使用 brew install git 或者直接下载安装包安装，如果提示需要安装 Command Line Tools 请先在 appStore 里下载 Xcode，然后进入 Preferences -> Download -> Command Line Tools -> Install 安装命令行工具。

    3. Linux

      Sudo yum install git-core 或者 apt-get install git
